**📝 项目描述 | Project Description**
中文
一款专为求职者打造的本地优先 AI 求职工具，一站式解决「找工作 + 改简历 + 管进度」全流程需求。核心功能包括精准岗位搜索、智能简历优化、求职进度管理，同时支持可选 AI 增强，无 API 时所有基础功能 100% 可用，兼顾隐私与体验。
English
A privacy-first, AI-powered job-hunting tool built exclusively for job seekers, streamlining the entire process of job searching, resume polishing, and application tracking. Core features include smart multi-platform job search, intelligent resume optimization, and visual application progress tracking, with optional AI enhancements that work fully without an API key, balancing privacy and user experience.
